# ACE Properties KC - Docs

This `/docs` folder is the single source of truth for the initial build of **acepropertieskc.com**.

## Contents

- `00-project-overview.md` - goals, audience, conversion actions, service area focus (including ZIP 64131)
- `01-brand-design-system.md` - logo direction, colors, typography, UI components
- `02-site-map.md` - IA, URL structure, required pages, component inventory
- `03-seo-strategy.md` - technical + on-page + local SEO plan; low-hanging keywords; ZIP targeting
- `04-copy-style-guide.md` - tone, messaging pillars, disclaimers, do/don't language
- `05-funnels-and-forms.md` - lead funnels, forms, CTAs, follow-up automation requirements
- `06-tech-stack-and-repo-setup.md` - recommended repo stack + libraries + setup commands
- `07-launch-checklist.md` - pre-launch and post-launch checklist

## Working style

We are building **docs-first** (MD) so we can:

1. Agree on scope, messaging, and compliance before building UI.
2. Keep implementation consistent across pages/components.
3. Move faster inside Windsurf by generating code from stable specs.
